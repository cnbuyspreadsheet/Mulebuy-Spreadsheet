## 🧪 QC Mindset

<details open><summary><strong>What should you check before shipping?</strong></summary>

- Compare warehouse photos against listing photos.
- Request measurements when sizing matters.
- Check embroidery, stitching, shape, color, tags, and hardware.
- Think about parcel weight before adding bulky shoes or jackets.
- Do not approve international shipping just because the item arrived at the warehouse.

</details>
